## Common Points of Measure (POM)

### Tops
- Chest width (1" below armhole)
- Body length (HPS to hem)
- Shoulder width (seam to seam)
- Sleeve length (shoulder to cuff)
- Sleeve opening
- Neck width
- Neck drop front
- Neck drop back
- Hem width
- Across front
- Across back

### Bottoms
- Waist (relaxed)
- Waist (stretched)
- Front rise
- Back rise
- Hip (at crotch)
- Thigh (1" below crotch)
- Knee (13" from crotch)
- Leg opening
- Inseam
- Outseam

### Dresses
- Bust
- Waist
- Hip
- Total length
- Skirt length
- Armhole depth

## Notes

- Measure each garment on a flat surface
- Do not stretch unless specified
- Round to nearest 0.5 cm / 0.25 inch
- Mark any out-of-tolerance measurements in red
